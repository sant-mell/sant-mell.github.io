# Portfolio Website

My personal portfolio website built using Next.js 16 (App Router), TypeScript, and Tailwind CSS v4. 

This site is designed to serve as a central hub for my resume, academic background at Tec de Monterrey, and hands-on projects, specifically tailored toward systems engineering, cybersecurity, and networking roles.

## Tech Stack
* **Framework:** Next.js 16 & React 19
* **Language:** TypeScript
* **Styling:** Tailwind CSS v4
* **Animations:** Framer Motion
* **UI Components:** Radix UI primitives / shadcn setup

## Project Highlights Included
* **DFA Lexer / Compiler:** A parallel syntax highlighter and compiler implementation in Python, Elixir, and Racket focused on automata theory.
* **MARIE.js:** A web-based assembly language simulator built with TypeScript to demonstrate instruction set architectures.
* **TC2005B Video Game:** An 8-credit flagship team project built in Unity/C# showing object-oriented programming and state management.

## Key Features
* **Interactive Timeline:** A custom radial-orbital visualization tracking my international academic background (Brazil, Netherlands, Mexico).
* **Live Activity:** Embedded telemetry displaying real-time GitHub contributions.
* **Performance:** Fully responsive layout optimized for fast loading speeds and dark mode support.
